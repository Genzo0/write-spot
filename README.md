## About The Project

This project was developed as part of a challenge given by Synapsis for their Front End Challenge. The goal of this challenge was to demonstrate my front-end development skills and ability to deliver a complete project within a limited timeframe.

- **Challenge Provider:** Synapsis
- **Duration:** 5 days
- **Objective:** To create a front-end application showcasing best practices in modern web development

I dedicated significant effort to ensure the project met all requirements and adhered to best practices. This included:

- **Planning:** Carefully analyzing the requirements and creating a plan to execute the project within the given timeframe.
- **Development:** Implementing the project using modern front-end frameworks and tools, focusing on clean, maintainable, and efficient code.
- **Testing:** Conducting thorough testing to ensure the application is robust and free of bugs.
- **Optimization:** Fine-tuning the application for performance and responsiveness.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
